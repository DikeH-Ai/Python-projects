ABOUT

This is a simple store-front website done in fufillment of the CS50 Course Assinment 8

The aim of this project is to create a store front that performs the following actions:

Initiate a conversation between a potential customer and an admin staff

Check out the Location of the store on a map with a click of a button

Sends Emails to store-front admins

And says the mobile number of the front
